# 工厂MES 使用说明

## 登录
浏览器打开网址 → 输入用户名密码 → 点击登录
- 管理员：admin / admin123
- 操作工：zhang3 / 123456

## 首页（生产看板）
- 顶部统计卡片：工单总数、生产中、待处理、今日完成
- 趋势图：近7天产量
- 仪表盘：完成率、合格率
- 快捷按钮：新建工单、报工、入库、检验

## 工单管理
- **查看工单**：卡片列表，显示进度、优先级
- **新建工单**：选产品、填数量、选产线和工艺模板 → 提交
- **工单详情**：点卡片进入，查看工序进度、质量记录、报工记录

## 车间报工（操作工主要用）
- **开始生产**：在工单卡片上操作 → 开始工序
- **报工**：填良品数、不良品数 → 提交
- **完成工序**：所有工序完成后，工单自动完成

## 库存管理
- **库存总览**：看所有物料库存数量，缺货标红
- **入库**：选物料、填数量、库位 → 入库
- **出库**：选物料、填数量、关联工单 → 出库
- **新建物料**：填名称、编码、单位等

## 质量管理
- **检验**：选工单、填检验结果（合格/不合格）、缺陷数量 → 提交
- **记录查看**：所有检验记录可追溯

## 系统管理（仅管理员）
- **用户管理**：添加/查看用户，分配角色
- **产品管理**：添加产品
- **工艺模板**：查看工序流程

## 手机打不开？
- 用 **Chrome 或 Safari** 打开，别用微信内置浏览器
- 首次加载可能等 **5-10秒**（数据库冷启动）
- 如果用移动网络打不开，换 WiFi 试试
- 如果都不行，给我发报错截图
